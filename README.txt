Description

Este proyecto contiene una serie de ejercicios enfocados en el uso de estructuras de datos en Python, específicamente listas y diccionarios. Se desarrollan funciones que permiten recorrer, manipular y extraer información de estas estructuras, reforzando la lógica de programación.

Funciones intermedias (Core)

En este conjunto de ejercicios se implementan funciones que trabajan con:

* Listas de diccionarios
* Diccionarios con valores tipo lista

Se aplican técnicas como:

* Iteración con bucles for
* Acceso a valores mediante claves
* Manipulación y actualización de datos
* Formateo de salida

Ejercicios incluidos

1. Iterar a través de una lista de diccionarios

   * Imprimir claves y valores de cada elemento

2. Obtener valores de una lista de diccionarios

   * Extraer valores según una clave específica

3. Iterar a través de un diccionario con listas

   * Mostrar cantidad de elementos y contenido por clave

Objetivos

* Poner en práctica la iteración de diccionarios y la creación de funciones
* Comprender mejor los recorridos de diccionarios de listas y listas de diccionarios
* Mejorar la lógica de programación en Python
* Preparar bases para el uso de librerías como pandas en análisis de datos
